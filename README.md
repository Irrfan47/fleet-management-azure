# FleetBuddy - Enterprise Vehicle Management

Advanced fleet management system built with React, TypeScript, and Laravel.

## Features
- Real-time Dashboard
- Vehicle & Driver Tracking
- Booking & Maintenance Management
- Insurance & Accident Logging
- Real Authentication (Laravel Sanctum)

## Setup
1. Backend: `composer install`, `php artisan migrate:fresh --seed`, `php artisan serve`
2. Frontend: `npm install`, `npm run dev`
